# Rig 버전별 가정 시나리오

모든 버전 공통: GoPro HERO5 Black ×2, 4K 16:9 29.97fps Wide EIS-off,
yaw 벌림각 68.8°(2026-07-12 세션 실측), 베이스 중앙 1/4"-20 너트 매입 마운트.
각 디렉토리의 `extrinsic.json`이 해당 STL과 쌍을 이루는 설계 공칭값이다
(세션별 자동 정합이 delta를 보정하는 전제).

커버리지 경험식: 사이드라인 근접 지점이 보이려면
`atan(높이/후퇴거리) ≤ pitch + 반수직화각(~35°)`.

| 버전 | pitch | baseline | 가정 시나리오 | 상태 |
|---|---|---|---|---|
| v1 | 18° | 99.3mm | 시험 세션 조준 재현. 위에서 눌러 넣는 스냅핏. 여유 있는 설치(높이 2m 기준 후퇴 1.5m+) | v2로 대체, 참고용 |
| v2 | 18° | 101.0mm | v1과 동일 조준. 뒤에서 밀어넣기 + 후면 개방(터치스크린 노출)으로 조작성 개선 | v4로 대체 |
| v3 | 18° | 71.6mm | v2 + 렌즈 안쪽 배치(왼쪽 카메라 roll 180°)로 baseline 29% 단축 — 선수가 심 근처로 접근하는 풋살 시차 완화 | v4로 대체 (v4 = v3+지붕) |
| v4 | 18° | 72.1mm | **표준 설치 권장.** v3 + 양쪽 지붕판(차양/빗방울/강성). 높이 2m → 후퇴 1.5m+, 높이 4m → 후퇴 3m+ 조건에서 근접 사이드라인까지 커버. 사각 반경 = 높이×0.75 | 권장 (표준) |
| v5 | 28° | 75.1mm | **사이드라인 여유 1m 경기장 전용.** 마스트 ~4m + 후퇴 ~1m 가정. 근접 커버 우선: 사각 반경 = 높이×0.5 (~2m), 하프라인 기준 사이드라인 ±1.8m 바깥부터 보임. 트레이드오프: 수평선 위 +7°까지만 담므로 높이 뜬 공은 잘릴 수 있음. 파노라마 출력 세로 범위도 함께 늘릴 것 | 권장 (근접 설치) |

주의: 높이 4m·후퇴 1m 조건에서 근접 교차점(내려보기 76°)과 반대편 상공을
동시에 담는 것은 16:9 세로 화각(~70°)으로는 불가능하다. v5는 근접 쪽을
우선한 절충값(pitch 28°)이다. 전 영역이 꼭 필요하면 2.7K 4:3 모드(세로
~94°) 전용 리그를 별도로 검토할 것 (해상도 손실 감수).
