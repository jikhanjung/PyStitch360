## analyze_video 파노라마 가정 점검 → reanalyze.py 오버라이드

기본값들이 초광폭 파노라마 튜닝임을 확인:

- `det_w` = 소스 폭/2 (파노라마 5906→2944 는 적절, **AX700 1920 은
  960 으로 과소** — 16:9 는 폭 전체 권장)
- `field_top_frac` 0.26 — 팬 카메라는 프레임 상단이 늘 관중석이 아님
- `far_boost` — 근측 뷰 소스엔 불필요

scripts/reanalyze.py 에 `--det-w / --field-top / --no-far-boost`
추가. AX700 권장 실행:
```
python scripts/reanalyze.py AX700.mp4 --det-w 1920 --field-top 0 --no-far-boost
```

## 조기 성과물: 하이라이트 대체 앵글 동시 추출

- `core/sync_multi.cut_synced_clip` — 시계 모델로 A 구간을 B 시각으로
  환산해 ffmpeg 재인코딩 컷 (-ss 선행 + 트랜스코딩 = 프레임 정밀,
  오디오 AAC 포함).
- 하이라이트 일괄 내보내기 대화창에 "동기화된 다른 카메라 앵글 동시
  추출" 체크박스 — `.events.json "sync"`(sync_cams.py 결과)가 있고
  상대 영상이 존재할 때만 활성. 구간별로
  `<이름>_h01_킥오프_<상대영상>.mp4` 가 함께 나온다. 실패는 구간별
  로그로만 (배치 전체를 죽이지 않음).
