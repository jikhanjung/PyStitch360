# 038: 검수 UI 개선 모음 (사용자 피드백 연속 반영)

- 날짜: 2026-07-19
- 종류: 구현 (037 이후 UI 피드백 일괄)

## 등번호 지정

- 우클릭 역할 지정 메뉴에서 각 팀 이름 아래 "↳ 팀 번호 지정" 서브메뉴:
  명단 항목 선택(✓=현재, (#tid)=이미 사용) / 직접 입력 / 명단 입력·수정
  (한 줄에 "7" 또는 "7 홍길동") / 해제.
- 번호는 병합 대표 tid 에 저장(.ptz.json "player_nums", 명단은
  "rosters") — 번호 지정은 팀 소속을 함의하므로 역할도 함께 맞춘다
  (GK 는 유지). 선수 목록엔 "팀1 7번", 오버레이 태그엔 숫자 표시.
- 역할 초기화 시 번호 지정도 초기화 (명단은 팀 이름처럼 유지).

## 기타 (커밋 단위 참고)

- 3행(영상/타임라인/정보) 세로 스플리터 — 높이 조절·저장 (d4cf2e6)
- PTZ 탭에서 하단 로그 자동 숨김 (d194b73)
- 레이더 show/hide 체크박스 + 불투명도 슬라이더 (6101dbf, f993b21)
