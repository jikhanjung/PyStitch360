# 037: 미리보기 오버레이 재구성 — 레이더 합성·표시 토글·로그 탭

- 날짜: 2026-07-19
- 종류: 구현 (사용자 요청 일괄)

## 레이더 → 영상 우하단 반투명 합성

별도 RadarView 위젯(하단 스트립)을 제거하고, 내보내기와 동일한
`draw_radar_panel` 을 미리보기 프레임 우하단에 addWeighted(0.55)로
합성 — 내보내기 결과와 같은 모습을 미리 본다. 캘리브레이션 전에는
카메라 기준 근사 좌표를 build_radar_data 와 같은 가정 위치로 이동해
같은 패널에 그린다. 팔레트는 역할 표시 색 그대로.

## 우상단 표시 토글 (영상 위 오버레이 체크박스)

"선수" (기존 체크박스 이동) / "공" / "크롭 박스" — 각각 show/hide:

- 공: 후보·수락·승격·무시 마커 + 수동 공 키프레임
- 크롭 박스: 계획 크롭 창 + 크롭 키프레임 화살표
- 상태는 QSettings 에 저장 (재시작 유지). 랜드마크 찍기 모드의
  전체 숨김은 종전대로 우선.

FramePane 위에 레이아웃을 얹어 영상 우상단에 반투명 배경으로 표시.

## 로그 탭

공/선수/경기장 탭 옆에 "로그" 탭 (QPlainTextEdit, 2000줄 순환).
PtzTab.log 가 메인 윈도우 로그와 탭 로그에 동시 기록 — 탭을 벗어나지
않고 분석/병합/렌더 메시지를 확인.

## 검증

컴파일 + pytest 93 통과. 레이더 위치/토글/로그 미러는 GUI — Windows
실기 확인.
