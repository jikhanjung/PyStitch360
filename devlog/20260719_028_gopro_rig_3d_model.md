# 028: 듀얼 GoPro 고정 리그 3D 모델 (2026-07-19)

## 배경

현재 사이드라인 촬영 리그는 Rode 콜드슈 분배 부품 + pitch/yaw 조절식 마운트로,
세션마다 방향이 달라져 캘리브레이션을 매번 다시 해야 한다. 측정된 캘리브레이션
값을 형상에 고정한 3D 프린팅 일체형 리그를 설계했다.

## 산출물 (`hardware/`)

- `rig_generator.py` — 파라메트릭 생성 스크립트 (trimesh + manifold3d, venv 사용)
- `rigs/<rig_id>/` — **버전 단위 리그 프로파일** (STL + 설계 extrinsic 쌍):
  - `rig.stl` — 일체형 리그 (베이스 + 쐐기 받침 + 크래들 ×2)
  - `test_cradle.stl` — 공차 확인용 단일 크래들 (쐐기 없음)
  - `extrinsic.json` — 설계 공칭 extrinsic (yaw/pitch/roll, baseline, 광학중심
    높이) + 카메라/촬영모드 메타데이터. Capture Profile의 시드로 사용,
    세션별 자동 정합이 delta를 보정하는 전제
  - `preview.png` — 3방향 렌더

### 리그 버전

- **GP5-DUAL-v1** (toploader): 위에서 눌러 넣는 스냅핏. 후면은 터치스크린
  창이 뚫린 프레임. 168×82×65mm, baseline 99.3mm
- **GP5-DUAL-v2** (rearload): 뒤에서 앞으로 밀어넣는 방식. 후면 완전 개방
  (터치스크린 노출), 좌우 상단 챔퍼 레일 + 바닥 스냅 탭이 뒤에서 걸어줌.
  전면은 스토퍼(하단 립 + 좁은 기둥 ×2). 171×85×67mm, baseline 101.0mm
- **GP5-DUAL-v3** (rearload + lens-inward): 시선은 v2와 동일한 ±34.4° 발산
  배치를 유지하되, **왼쪽 카메라를 roll 180° 뒤집어 장착**해 모서리에
  치우친 렌즈 두 개를 모두 안쪽 변으로 모음. 뒤집힌 크래들은 받침대를 높여
  렌즈 높이를 정확히 일치시킴(dz=0). 크래들 간격도 회전된 실제 형상 기준
  최소 패킹으로 축소. **baseline 101.0 → 71.6mm (-29%)**. 168×85×82mm.
  - HERO5 렌즈 위치 실물 확인됨: 후면 기준 왼쪽(전면 기준 오른쪽) →
    안쪽 배치를 위해 뒤집는 쪽은 왼쪽 카메라
  - 뒤집힌 카메라: GoPro 회전 설정(Down)으로 영상은 정방향 기록됨.
    셔터 버튼이 아래를 향하므로 녹화 시작은 터치스크린/음성/앱 사용
- **GP5-DUAL-v4** (v3 + 지붕): 양쪽 크래들 상면에 3mm 지붕판 추가
  (차양/빗방울 방지 + 강성 보강). 셔터 버튼 위치에 36×20mm 손가락 구멍.
  뒤집힌 크래들은 지붕판이 바닥이 되어 쐐기와의 접합면도 넓어짐.
  baseline 72.1mm, 169×87×82mm

## 설계 사양

- **각도 (프로젝트 캘리브레이션 데이터 기준)**
  - yaw 벌림각 68.8° (project JSON들의 `yaw_split_deg` 68.68~68.83° 평균) → 카메라당 ±34.4°
  - pitch 18° 하향 (devlog P01의 평균 시선 하향각)
  - 광축 간격 ≈ 77mm (목표 ≤100mm 충족)
  - 스크립트 상수로 노출되어 있어 변경 후 재생성 가능
- **크래들**: HERO5 Black 맨몸(프레임 없이) 62.3×44.9×24.6mm + 0.4mm 공차
  - 상면 개방 (셔터 버튼), 양 측면 개방 (모드 버튼 / USB 도어, 개구부 ≈20.5mm)
  - 전면은 렌즈 아래 12mm 립만, 렌즈 돌출부는 자유 (전면 기둥 폭 3.5mm로 렌즈 범프 간섭 회피)
  - 후면은 터치스크린 창(테두리 6mm)이 뚫린 프레임
  - 삽입: 뒤쪽 상단 립 아래로 밀어 넣고 앞을 눌러 스냅 (기둥 상단 경사 돌기 1mm)
  - 바닥은 막힘 → 배터리 교체 시 카메라 분리 필요 (USB 급전 운용 전제)
- **마운트**: 베이스 중앙 1/4"-20 관통홀 + 바닥면 육각너트 매입 포켓
  (강철 너트가 하중을 받으므로 프린팅 나사산 불필요)

## 소재/출력 권장

- PLA 부적합: 여름 야외(직사광선)에서 유리전이 ~60°C 근처, 크리프로 각도 틀어짐
- **PETG 또는 ASA 권장** (ASA가 UV 내성 최상), 퍼리미터 5~6, 인필 50%+
- 하중 자체는 작음 (카메라 2×118g, 정적) — 관건은 강성/내열/내충격이지 강도가 아님

## 방향 (Capture Profile)

리그를 "제품"처럼 버전 관리: `rigs/<rig_id>/`가 STL과 캘리브레이션 초기값의
쌍이 되는 단위. 향후 `extrinsic.json`을 확장해 intrinsic 참조, warp map,
blend mask, 경기장 프리셋까지 묶은 Capture Profile로 발전시키고, GUI에서
"Rig 선택 → 프로파일 자동 적용" 흐름으로 연결 예정. baseline이 현재 설계상
~100mm로 목표 상한이므로, v3에서 렌즈를 안쪽으로 모으는 배치(크래들 미러링/
피벗 위치 변경)로 단축 검토.

## 미확정/검증 필요

- 렌즈 좌우 위치 미확정 → 대칭 설계로 회피했으나, 간섭 시 슬라이서에서 미러링
- 본체 실측 공차: `test_cradle.stl` 먼저 출력해 밀착도 확인 후 `FIT_CLR` 조정
- 삼각대 장착 방식은 1/4"-20 단일 볼트로 시작, 회전 방지 필요 시 보스 추가 예정
