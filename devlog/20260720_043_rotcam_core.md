## core/rotcam.py (신규)

AX700 = 고정 위치 삼각대 + 팬(+드문 줌)의 표준 원근 카메라:

- `calibrate_reference` — 기준 프레임 랜드마크 → f(1D 황금분할 탐색 ×
  평면 PnP)·자세·설치 위치. 기존 파노라마용 fit_field_calibration 은
  원통 모델이라 재사용 불가 — 원근 모델 별도.
- `decompose_H` — 프레임 간 호모그래피 H = K′RK⁻¹ 를 줌 비율 1D 탐색
  + 정규직교 잔차 최소화로 분해, R 은 SVD 최근접 회전. **줌이 추적
  파라미터로 흡수** (P06 설계 그대로).
- `track_step` — RANSAC H + 분해 + 잔차 게이트(가림·블러 기각) 체인.
- `anchor_rotation` — **누적 드리프트 제거의 핵심**: 설치 위치 고정
  조건에서 현재 프레임 랜드마크의 world 광선 ↔ 카메라 광선을
  Kabsch(직교 프로크루스테스) 정렬 → (R, f) 절대 재추정.
- `pixel_to_field` / `field_to_pixel` — 지면(z=0) 광선 교차 투영.
- `match_frames` — SIFT 매칭 유틸 (실영상 추적 입력).

## 합성 검증 (tests/test_rotcam.py — 통과)

GT: 반대편 사이드라인 (3, 45, 5.5m), f=1400/1920px, 노이즈 0.5~1px.

- 기준 캘리브레이션: f 오차 <1%, 설치 위치 <0.5m.
- 단일 스텝 분해: 팬·팬+줌(±5%) 회전 오차 ≤0.18°, 줌비 소수 4자리.
- **체인만으론 랜덤워크 누적** (80스텝 → ~2° → 지면 수십 m) — 계획의
  "주기 재정렬" 필요성을 정량 확인.
- 20스텝마다 anchor_rotation → 줌 구간 포함 80스텝 후 **지면 투영
  오차 <1.5m, f <2%** (P06-3 완료 기준 "rms 수 m" 충족 경로).
- 무상관 매칭·매칭 부족 기각 (품질 플래그 경로).
