# 025: 공중볼 3D — 탄도 피팅으로 레이더 XY 직선화

- 날짜: 2026-07-19
- 종류: 작업 기록
- 관련: [024](20260719_024_seed_propagation.md), 사용자 관찰 (레이더 XY 포물선)

## 모델 (core/airborne.py)

카메라(높이 h)가 높이 z 의 공을 보면 지면 투영은 카메라 반대쪽으로
배율 h/(h−z) 만큼 밀린다 — 공중볼이 레이더 XY 에서 포물선으로 휘는
원인. 탄도 모델(XY 등속 직선 + Z 중력 포물선, 파라미터 p0·v·t0·T)을
관측 지면 궤적에 LM 피팅해 참 XY 와 z(t) 를 복원한다.

- `detect_airborne_segments`: 유한 런을 0.5~3.5s 창으로 훑어, 탄도가
  등속 직선 대비 잔차 1.8× 이상 개선 + 정점 0.8m~0.85h + 속도
  4~32m/s 인 비겹침 구간 채택. (속도 상한은 실측에서 나온 45~66m/s
  비물리 체인 오검출을 기각하기 위해 추가.)
- 한계: z ≥ h 면 시선이 수평선 위 — 카메라 높이(4.1m)를 넘는 높은
  공은 지면 투영 자체가 사라져 보정 대상이 아님 (검출 공백으로 나타남).

## 통합

- **편집 레이더**: 링크/수락 재계산 시 `_recompute_airborne` 가
  {si: (X, Y, z)} 캐시 생성, _redraw 의 공 위치가 보정 XY 사용.
  공중 구간·정점 높이는 로그로 표시.
- **내보내기 미니맵**: build_radar_data 가 캘리브레이션 있을 때 공
  트랙에 같은 보정 적용.

## 검증

- 합성 4종: 투영 배율 정확(h/(h−z) 1% 이내), 피팅 복원(t0/T/정점/속도),
  감지+직선화(관측 휨 0.8m+ → 보정 0.35m 미만, 참 궤적 0.5m 이내),
  굴림 공 비오인.
- 실측: 공중 구간 **36개** (속도 중앙 14·최대 31m/s, 정점 중앙 1.1m·
  최대 3.2m — 유소년 경기로 그럴듯). 대표(23:25) 시각화: 관측 지면
  궤적 25m 돌출이 보정 후 짧은 직선 + z 2.1m 상승으로 복원.
