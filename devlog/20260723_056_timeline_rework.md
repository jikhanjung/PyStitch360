# 056: 타임라인 개편 — 서브행·번호 행·접기·꽉 채움

- 날짜: 2026-07-22 ~ 07-23
- 종류: 구현 (UI)

- **공 서브행**: 동시 공 트랙(경기 공 + 옆 구장)이 한 줄에 포개지던
  것을 선수 레인과 같은 그리디 간트로 분리. 히트도 서브행 우선.
- **번호 행** (사용자 방향): 번호가 부여된 트랙릿은 번호별 전용 행
  으로 맨 위 — 같은 번호 조각이 한 줄로 이어지고, 부여 즉시 재배치.
  흰 테두리 + 높이 되면 번호 표시. 행의 빈 구간 = 그 선수를 놓친
  시간이라 검수 대상이 시각화된다.
- **레인 접기/단독 확대**: 우클릭 메뉴 — 접힌 레인은 12px 한 줄
  (서브행 겹침), "이 레인만 확대" 는 나머지 전부 접고 유연 공간
  독차지 (토글 복원). QSettings 지속.
- **꽉 채움**: 고정 높이 → 패널 높이를 항상 채움 (lane_h 는 비율,
  접힘은 고정). 스플리터로 키우면 레인이 같이 커진다.
- 부수: 소유 리본(P08) 팀 색 밴드, 앵글 레인 클릭 = 카메라 전환.
