# 052: 지표 UI 일괄 — 패스맵·뛴 거리·소유 리본·alt 오버레이

- 날짜: 2026-07-21
- 종류: 구현
- 관련: [P08](20260721_P08_match_metrics_plan.md) 3단계,
  [P07](20260721_P07_pitchwatch_multicam_ui.md) 2단계 일부

1. **패스맵 렌더** (core/metrics.render_passmap): 필드 위 노드(선수
   평균 위치, OCR 등번호) + 화살표(두께=횟수), 팀별 1장 — 통계 창
   스크롤 행에 표시. mean_positions 는 검출 30샘플 미만 제외.
2. **뛴 거리 표**: 리포트의 movement_stats 재사용 (0.5s 리샘플·속도
   상한). 각 행에 관측 비율 병기 — 커버리지 60% 의 6km 와 95% 의
   6km 를 구분해 읽게 (P08 정직성 원칙).
3. **소유 리본**: 지표 계산 결과의 소유 구간을 눈금자 아래 팀 색
   4px 밴드로 — v1 은 표시만.
4. **alt 검출 오버레이** (P07-2 일부): 디코더가 t_alt 를 함께 반환,
   alt 의 .analysis.json 을 지연 로드해 최근접 샘플(±0.5s)의 선수
   박스/공을 픽셀 좌표로 표시 — 세 모드 공통, C0011/C0012 가 첫
   실데이터.

스위트 131개 + offscreen 스모크 통과. 실경기 확인은 Windows 캘리브
레이션 후 — 통계 창까지 한 클릭 경로가 완성된 상태.
