# 044: 리그 v6 — 발열 대응 캐노피 (8K 대비)

- 날짜: 2026-07-20
- 종류: 하드웨어 (rig_generator)
- 관련: [028](20260719_028_gopro_rig_3d_model.md) (v1~v4),
  [P06](20260720_P06_multicam_fusion_plan.md) Ace Pro 2 발열 체크리스트

## 설계 (GP5-DUAL-v6 = v5 조준 + 열 캐노피)

v4/v5 의 밀착 지붕은 햇빛을 받아 카메라 상면으로 전도·복사되고 대류를
막는다 — 8K급(Ace Pro 2 후보)은 발열 스로틀링이 관문이므로:

- **18mm 공기층**: 레일 위로 측벽을 올리고 얇은 판(2.4mm, 저열용량)을
  띄움 — 전후·좌우 개방으로 대류 유지.
- **처마**: 외측/후방 12mm, 전방 6mm (렌즈 후드 겸용), 내측 2mm
  (v3식 밀착 패킹 유지).
- 상승기류 벤트 슬롯 2개 + 셔터 손가락 구멍(상단 캐노피만).
- **좌측 크래들은 바닥 아래 캐노피**: lens-inward 배치에서 좌측
  카메라가 뒤집혀 장착되므로, 로컬 상단 캐노피는 뒤집힌 뒤 바닥으로
  간다 — 로컬 바닥 밑에 달아야 하늘을 향한다 (v4/v5 의 지붕도 좌측은
  사실 바닥이 하늘을 보던 구조였음). build_rig_lens_inward 에
  cradle_left 파라미터 추가로 좌우 비대칭 크래들 지원.
- 일체형 (사용자: 한여름용이라 탈착식이 이상적이나 일단 일체형 OK) —
  레일 슬라이드-온 탈착 변형은 후속.

## 검증 (생성 로그)

- **FOV 간섭 검사 신설**: 캐노피 판 둘레를 렌즈에서 샘플링 — 수평
  화각(±65° 패딩) 안의 점만 수직 반화각(35°) 이상 여유 요구. 상단
  61.1° / 하단 68.4° 여유로 통과. (초기 단순 검사가 측면 모서리를
  과보수적으로 걸러 이 형태로 정교화 — 검사가 실제로 전방 처마 12→
  6mm 축소를 이끌었음.)
- 크래들 교차 검사를 크래들끼리로 한정 — 캐노피로 커진 bbox 때문에
  웻지 풋프린트가 중심선을 살짝 넘지만 union 되므로 무해(보강 효과).
- optical check: yaw split 68.80°, pitch 28°, baseline 74.4mm,
  렌즈 높이차 0. rig 211.8×115.0×104.1mm, watertight.
- v1~v5 전체 재생성 회귀 — 기존 수치·파일 불변.
