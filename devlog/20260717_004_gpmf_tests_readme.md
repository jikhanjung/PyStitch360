## GPMF 자이로 파싱 (`core/gpmf.py`)

ffmpeg 으로 gpmd 트랙 추출 → KLV 파싱 (SCAL 스케일 적용, GYRO int16 ×3,
~397Hz). 각속도 스파이크(기본 1.0 rad/s)를 5초 간격으로 그룹핑하고,
이벤트 창 ±2초의 순 회전량(∫ω dt)으로 "방향변경 / 일시흔들림" 분류.

**실데이터 검증 결과가 모델을 실증**: 삼각대 정지 상태 자이로 잡음
중앙값 0.011 rad/s. 이벤트는 녹화 시작 직후(0:01 / 0:18 / 0:38 —
설치 조작)와 종료 직전(39:15 — 회수)에만 존재, **경기 중 38분간 0건**.
piecewise-constant 방향 모델이 실제 촬영 조건과 정확히 일치.

GUI: 정합 탭 세그먼트 박스에 "자이로에서 충격 이벤트 탐지" 버튼 + 후보
목록. 더블클릭하면 이벤트 3초 뒤(안정화 시점)로 이동 → 거기서 재정합하면
새 세그먼트.

## README 재작성

구버전(360° equirect) 설명을 현재 아키텍처로 전면 교체: 파이프라인 다이어그램,
기능 목록, 설치/사용법, 촬영 가이드(HERO5, EIS off, 밀착 마운트), 렌즈 프로파일
추가 방법, 저장소 구조.

## 코어 유닛 테스트 (`tests/test_core.py`, 12개)

- chapters: 챕터 체인 정렬(GP10 포함), 비 GoPro 파일, 디렉터리 그룹핑
- geometry: 픽셀↔광선 왕복(오차 <0.05px), Kabsch 정확 복원,
  RANSAC 20% 아웃라이어 내성
- render: 심 가중치 이진성/마스크 폴백
- project: Alignment 직렬화 왕복, 절대경로 깨졌을 때 상대경로 복구
- sync: ffmpeg 합성 노이즈로 1.5초 오프셋 재현 (오차 <20ms)
