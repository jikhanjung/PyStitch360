## 갭필 2차 패스 (구현 + 실측)

- core `gapfill_targets`(수락 트랙 사이 ≤4s 갭의 보간 목표) +
  `gapfill_analysis`(보간점 중심 640 타일, conf 0.06, **공+사람 동시
  검출**). 공: 반경 250px 내 최고 후보 주입 — 저장 conf 바닥 0.26
  (링크 문턱 통과), 원래 conf 6번째 원소 보존(길이 6 = 갭필 행).
  사람: 중복 아닌 것만 id -1 + 상반신 HSV 로 주입.
- CLI scripts/gapfill.py (백업 .analysis_pregapfill.json, 전후 비교),
  GUI 분석 메뉴 "갭필 2차 패스" (GapfillWorker, 진행률/취소).
- **시크 최적화**: 갭 목표는 연속 샘플 — 랜덤 시크(~1.4s/지점) 대신
  90프레임 이내면 순차 grab. 0.7 → **14.3지점/s (20×)**.
- 실측(match_20260712, 목표 5,928): 공 1,046·선수 5,743 주입, 7.1분.
  **수락 21.2→25.5% (시간 15.4→17.2분), 트랙 219→195개** (파편 병합).

## 호각 검출기 (core/audio.py)

이벤트 엔진(킥오프 등)의 청각 신호. 전체 타임라인을 **연속 트랙**으로
`<pano>.whistle.json` 에 보관 (band_db·tonal, 0.1 양자화, 32ms 해상도)
— 이벤트는 트랙 위에서 파생, 문턱 변경 시 재추출 불필요.

- 1차 시도(광대역 대비 대역 비율)는 실패 — 야외 녹음은 관중·바람이
  광대역을 지배해 호각 대역 비율이 항상 음수(p50 −24dB).
- **프로미넌스 방식**: 3.2~4.8kHz 대역 에너지의 4s 롤링 중앙값 대비
  상승분. 실측에서 호각 +24~31dB vs 잡음 p99 13.5dB — 깨끗한 분리.
  히스테리시스(15/8dB) + 피크 협대역성(tonal≥6, 박수 같은 광대역
  충격음 기각) + 병합(0.25s, 삑-삑 이중 호각).
- 실측: 39.5분에서 **93개 이벤트**, 앵커 일치 — 0:17.3 **1.34s 롱
  휘슬(킥오프)**, 39:16.8/27.5/30.6 종료 삼중 호각. 추출 16초 (CPU).
- 타임라인에 **호각 레인** 추가: 프로미넌스 세로 바(6dB~, 15dB+ 주황)
  + 이벤트 틱. GUI 는 파노라마 열 때 .whistle.json 자동 로드.
- 테스트 5개 (합성 호각 검출·시각, 이중 호각 병합, 비호각 기각,
  트랙 커버리지, 저장 라운드트립). 전체 61개.
